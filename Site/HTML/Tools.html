Unity

Unity is a cross-platform game engine developed by Unity Technologies and used to develop video games for PC, consoles, mobile devices and websites. Unity Pro is available for a fee and Unity Personal has no fee; it is available for any use to individuals. Unity is noted for an ability to target games to multiple platforms. Five versions of Unity have been released.

Usequencer

uSequencer is a powerful and flexible Event based sequencer for Unity, written entirely in c#. uSequencer is incredibly flexible, and ships with a large selection of Built-In Events. It is designed the way they look and feel. uSequencer is built entirely into the editor, utilising the inspector and rendering into it's own window. You can do everything, right through the interface. It is Easy, Intuitive, Unobtrusive.
